Pet computação é um programa do ICMC-USP responsável por fortalecer a integração entre a universidade e a comunidade. Esses dois projetos visam ensinar lógica de programação à alunos de escola pública em São Carlos - SP, como monitora minhas responsabilidades são preparar e ministrar aulas, auxiliar os alunos durante as aulas e propor atividades para serem feitas em aula

Monitoria Codifique

Esse projeto é voltado para alunos de ensino médio (15~18 anos), usamos python para ministrar as aulas no curso.
